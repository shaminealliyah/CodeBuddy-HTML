HTML Example: User-Friendly Color Picker
<!DOCTYPE html>
<html>
<head>
<title>Color Picker - User Friendly Design</title>
</head>
<body>

<h1>Choose Your Favorite Color</h1>
<label>Select a color: </label>
<input type="color" id="colorPicker" value="#ff66b3">
<br><br>
<div id="colorBox" style="width:200px; height:200px; background:#ff66b3; border:2px solid black;"></div>

<script>
  document.getElementById("colorPicker").oninput = function() {
    document.getElementById("colorBox").style.backgroundColor = this.value;
  };
</script>

</body>
</html>
