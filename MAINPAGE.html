About this course

Fun fact: all websites use HTML — even this one. It's a fundamental part of every web developer's toolkit. HTML provides the content that gives web pages structure. By using elements and tags, you can add text, images, videos, forms and much more. Learning HTML is the first step in your web development journey and an essential skill for both front-end and back-end developers.
